# cs458-Project2

## Requirements
- Android Studio
- Android-Sdk-29
- Appium
- Java
- Npm

## How to start Project
- Open Android Studio
- Install Gradle Dependencies
- Run application to install the app to your phone.
- Open the tests project. 
- Enter your phone credentials in the config part in Main.java
- Run the program.
- Enjoy the test.
